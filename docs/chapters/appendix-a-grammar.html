Appendix A: Grammar Reference

This appendix is a quick reference for Zebra's syntax. For detailed explanations, refer to the chapter listed for each feature.

Basic Types

Primitive Types

var x as int = 42           # 64-bit integer

var y as float = 3.14       # 64-bit floating point var b as bool = true        # Boolean var s as str = "hello"      # Immutable string var c as char = 'x'         # Single character

References: Chapter 02 (Values and Types)

Nullable Types

var x as int? = nil         # Can be int or nil

var s as str? = "hello"     # Can be string or nil


if x != nil     println(x)              # Safe to use x here

References: Chapter 11 (Nil Tracking and Safety)

Collection Types

var numbers as List(int) = List(int)()

var mapping as HashMap(str, int) = HashMap(str, int)() var unique as Set(str) = Set(str)()


numbers.add(1) mapping.put("key", 42) unique.add("value")

References: Chapter 03 (Collections)

Result Types

var result as Result(int, str) = Result.ok(42)

branch result     on ok(value)         println(value)     on err(error)         println("Error: ${error}")

References: Chapter 12 (Error Handling with Results)

Functions

Basic Function Definition

def greet(name as str) as str

    return "Hello, ${name}"


var message = greet("Alice")

References: Chapter 04 (Functions and Scope)

Functions with Multiple Parameters

def add(a as int, b as int) as int

    return a + b


var sum = add(3, 4)

Functions with Optional Parameters

def calculate(a as int, b as int, operation as str) as int

    if operation == "add"         return a + b     else         return a - b

Returning Multiple Values (via Structure)

class Result

    var value as int     var success as bool


def divide(a as int, b as int) as Result     var result = Result()     if b != 0         result.value = a / b         result.success = true     else         result.success = false     return result

var person = Person("Alice", 30) println(person.describe())

Error Handling

Result Type

def divide(a as int, b as int) as Result(int, str)

    if b == 0         return Result.err("Division by zero")     return Result.ok(a / b)

Checking Results

var result = divide(10, 2)

if result.isOk()     println(result.value()) elif result.isErr()     println(result.error())

Unwrapping Results

var result = divide(10, 2)

<h1>Unwrap with default</h1> var value = result.unwrapOr(0)  # 0 if error


<h1>Unwrap or panic</h1> <h1>var value = result.unwrap()  # Crashes if error</h1>

List Operations

HashMap Operations

for key in map.keys()     var value = map.fetch(key)     if value != nil         println("${key} => ${value}")

Common Patterns

Safe Navigation

Try-Catch Pattern

var result = operation()

if result.isErr()     println("Error: ${result.error()}") else     println("Success: ${result.value()}")

    println(item)
